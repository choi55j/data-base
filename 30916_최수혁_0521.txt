☆ update문 : 테이블에 있는 데이터 수정하기

update 테이블명
set 변경할 열1 = 데이터, 변경할 열2 = 데이터, .... 변경할 열n 데이터
where 데이터를 변경할 대상 행을 선별하기 위한 조건


문제31> min_salary 값 전체를 9870으로 바꾸시오

SQL> update new
  2  set min_salary=9870;

문제32> 새로운 테이블에서 job_title이 Manager로 끝나는 max_salary 값을 7770으로 수정하시오.

SQL> update new
  2  set max_salary = 7770
  3  where job_title like '%Manager';


☆ delete문 : 테이블에 있는 데이터 삭제하기

delete from 테이블명
where 삭제할 대상 행을 선별하기 위한 조건식;



문제33> 새로운 테이블 new에서 job_title이 Manager로 끝나는 행을 삭제하시오

SQL> delete from new
  2  where job_title like '%Manager';
